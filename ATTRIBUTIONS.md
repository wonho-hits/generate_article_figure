## License notes

- **CC BY 3.0** (Servier Medical Art): requires attribution. The Servier icons
  are line-art biological symbols originally published by Servier
  Pharmaceuticals as a public scientific resource and aggregated by
  bioicons.com.
- **CC0 1.0** (Xi Chen): public-domain dedication; no attribution legally
  required. We credit anyway as a courtesy and to make provenance auditable.
- The `chromosome` icon has had its baked-in "Allele 1/2/3" labels and
  connecting arrows removed during sanitisation; otherwise the geometry is
  unchanged.

## Adding more icons

Use `scripts/sanitize_bioicons.py`:

1. Add an entry to `PICKS` in that script with the slug, repo path, author,
   license, and whether to strip baked-in labels.
2. Run `uv run python scripts/sanitize_bioicons.py` to regenerate
   `app/domain/_bioicons_data.py`.
3. Add a matching `SymbolEntry(...)` to the `CATALOG` in `app/domain/bio_symbols.py`
   (category `"cell_division"` or a new appropriate category).
4. Update this file with the new attribution row.
